# PRIME – Power Regulation and Intelligent Monitoring Equipment

## Overview

PRIME is a multidisciplinary engineering project developed under the Product Development Laboratory, Department of Electrical Engineering, NIT Rourkela.

The system integrates electrical power monitoring, protection mechanisms, embedded systems, cloud connectivity, and a mobile application into a unified intelligent platform.

---

## Objective

* Monitor real-time electrical parameters (voltage, current, power)
* Provide intelligent protection against overloads and faults
* Enable remote control of electrical loads
* Deliver insights through a mobile application interface

---

## System Architecture

The PRIME system consists of four major domains:

1. **Hardware Layer**

   * Voltage and current sensing circuits
   * Relay-based load control
   * Protection mechanisms (MCB, fuse, MOV)

2. **Embedded System**

   * ESP32 microcontroller
   * Data acquisition and control logic

3. **Cloud Layer**

   * Data storage and processing
   * API for communication
   * Authentication and security

4. **Application Layer**

   * Android mobile application
   * Real-time data visualisation
   * Remote control and alerts

---

## Key Features

* Real-time power monitoring
* Historical data analysis
* Remote ON/OFF load control
* Fault detection and alerts
* Secure cloud-based communication
